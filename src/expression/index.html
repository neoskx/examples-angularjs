AngularJS Exmaple - Expression

Expressions are JavaScript-like code snippets that are usually placed in bindings such as {{ expression }}

Angular expressions and Javascript expressions with following differences:

  • Attribute Evaluation: evaluation of all properties are against the scope, doing the evaluation, unlike in JavaScript where the expressions are evaluated against the global window
  • Forgiving: expression evaluation is forgiving to undefined and null, unlike in JavaScript, where trying to evaluate undefined properties can generate ReferenceError or TypeError.
  • No Control Flow Statements you cannot do any of the following in angular expression: conditionals, loops, or throw. Open and read Javascript Control Flow
