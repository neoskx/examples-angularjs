AngularJS Exmaple - Binding

The ngBind attribute tells Angular to replace the text content of the specified HTML element with the value of a given expression, and to update the text content when the value of that expression changes.

  1. 1+1 = {{1+1}}
  2. 2*3 = {{2*3}}
  3. Name: {{'Blithe'+ ' Xu'}}
  4. {{testString}}
  5. {{'Hello, '+ who}}
It is preferrable to use ngBind instead of curly markup. If you use curly markup, when AngularJS compile time, user will see markup.
